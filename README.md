στον ίδιο φάκελο . Η εκπαίδευση του λεξικού θα πάρει αρκετή ώρα (περίπου 15 λεπτά) διότι το εκαπαιδεύουμε σε όλα τα overlapped patches τα οποία είναι περίπου 63000 .
